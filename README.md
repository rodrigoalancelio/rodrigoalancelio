# 💼 Sobre mim

Minha jornada na tecnologia começou pela curiosidade de entender como os computadores funcionavam. O que era apenas um curso de hardware acabou se tornando uma carreira de mais de 10 anos em Infraestrutura de TI.

Hoje atuo como **Analista de Infraestrutura**, trabalhando com ambientes corporativos, administração de servidores, redes, segurança, Microsoft 365, Google Workspace, monitoramento, automações e melhoria contínua dos processos de TI.

Além da atuação corporativa, desenvolvo projetos próprios através da **RA TECH Solutions**, criando ferramentas voltadas para gestão, automação e inteligência artificial.

Meu objetivo é construir soluções que reduzam trabalho operacional e permitam que a TI seja cada vez mais estratégica para o negócio.
